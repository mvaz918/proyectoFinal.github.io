Sobre el proyecto

Este sitio web fue desarrollado como parte de un proyecto académico para presentar dos experiencias personalizadas dirigidas a jóvenes interesados en estudiar Diseño de Medios Interactivos en la USFQ.

A través de dos recorridos, cada uno pensado para un perfil específico (buyer persona), se abordan las etapas del proceso de decisión: atracción, interés, decisión y acción. Cada uno incluye contenido adaptado a sus motivaciones, intereses y dudas comunes, con recursos como publicaciones, testimonios, formularios y más.

El sitio está optimizado para buscadores (SEO), cuenta con código de seguimiento de Google Analytics para analizar el tráfico, y está integrado con un CRM para capturar datos de contacto e implementar estrategias de email marketing.
